Avant de commencer
	Installation de WindowBuilder:
		- Tout dépendamment de votre version d'Eclipse, suivre les instructions à cette adresse:
			http://www.eclipse.org/windowbuilder/download.php

Importation du projet
	- Ouvrir Eclipse, et créer/sélectionner un 'workspace'.
	- Aller à 'File' -> 'Importer'.
	- Sélectionner 'General' -> 'Existing Projects into Workspace' puis 'Next'.
	- Sélectionner le dossier dézippé du travail dans 'Select root directory'.
	- Sélectionnez le projet 'GLO3004_TP2' dans l'encadrer 'Projet', puis 'Finish'.

Exécution du projet
	- Sélectionner 'MainWindow.java' qui se trouve sous les packages 'src -> ui'
	- Exécuter l'application avec la commande 'Run' dans le menu 'Run'. (CTRL+F11)

Comment ça marche?
	Démarrer une simulation
		- Entrer les configurations désirées dans l'interface.
		- Une fois les configurations choisies, appuyer sur 'Start' pour démarrer la simulation.
		- Les résultats obtenus sont affichés dans la console d'eclipse.

	Mettre en pause une simulation
		- À tout moment, vous pouvez appuyer sur 'Pause' pour mettre en pause la simulation.
		- Réappuyer sur le même bouton ('Unpause') pour reprendre la simulation.

	Arrêter une simulation
		- Appuyer sur le bouton 'Stop' pour arrêter une simulation.
		** ATTENTION : La simulation sera complêtement perdue.

Bonne simulation!
